# Instalación

Copia la carpeta `.claude` en la raíz de tu proyecto Laravel.

Ruta final:

```text
<proyecto>/.claude/skills/laravel-clean-architecture/SKILL.md
```

Claude Code la detectará como una Skill del proyecto. Puedes invocarla manualmente con:

```text
/laravel-clean-architecture <tarea>
```

Ejemplos:

```text
/laravel-clean-architecture diseña el módulo de partidas de bingo
/laravel-clean-architecture revisa este Action y elimina sobrearquitectura
/laravel-clean-architecture crea el flujo para extraer bolas sin duplicados
```
